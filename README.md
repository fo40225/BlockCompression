## BlockCompression library

This library contains high-performance versions of the following compression algorithms:

* zlib
* QuickLZ
* Zstd
* snappy
* lz4

## Building

### Windows

Compile using BlockCompression.sln in Visual Studio 2017

### Linux

```
make -j
```

### Mac OS X

```
make -j -f Makefile.MacOSX
```
